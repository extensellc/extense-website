Open Positions

Senior DITA Information Architect

Remote / Hybrid  |  Full-Time  |  $120K–$160K

Apply Now

Design complex reuse strategies and taxonomy maps for Fortune 500 clients. Define specialization models, SubjectScheme maps, and content governance policies. Must have 5+ years with DITA-OT, Oxygen XML Editor, and at least one enterprise CCMS (IXIASOFT, Heretto, or Tridion Docs).

Required: DITA 1.3, XSLT 2.0, Git, Schematron. Preferred: S1000D, Lightweight DITA, CI/CD pipelines.

XSLT / Publishing Developer

Remote  |  Full-Time  |  $100K–$140K

Apply Now

Build PDF and HTML5 publishing plugins for the DITA Open Toolkit. You'll write XSL-FO templates, customize XHTML post-processing, and automate multi-format builds via Ant/Gradle scripts.

Required: XSLT 2.0/3.0, XSL-FO, Ant, Java (basic). Preferred: DITA-OT plugin architecture, FOP/AH Formatter, Jenkins.

Technical Writer — Life Sciences

Remote  |  Contract (12 months)  |  Competitive Rate

Apply Now

Author Instructions for Use (IFU) for Class II/III medical devices targeting 30+ markets. You'll work within a validated DITA workflow (21 CFR Part 11) and coordinate with regulatory affairs teams on EU MDR compliance.

Required: Medical device documentation experience, DITA, regulatory submission familiarity. Preferred: SPL, eCTD, ISO 14971.

CCMS Solutions Consultant

Remote / Travel 25%  |  Full-Time  |  $110K–$150K

Apply Now

Lead CCMS vendor selection, configuration, and deployment projects. You'll run PoC evaluations, configure workflows and permissions, and manage post-go-live optimization. Deep knowledge of at least 2 CCMS platforms required.

Required: IXIASOFT, Heretto, Paligo, or Tridion Docs experience. Preferred: XML database administration, SSO/SAML, REST API integration.
